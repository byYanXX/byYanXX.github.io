Vim 快捷键

2023-02-18

前言

Vim 模式：

  • 普通模式：光标移动时所在的模式，按 <ESC> 即可转到该模式
  • 插入模式：在普通模式下，按 i、a 或 o 之后，即可编写文本
  • 可视模式：按下 v，有点像 Shift 多选，常配合 y、p 实现复制内容、粘贴内容
  • 替换模式：按下 R，连续替换光标处的文本
  • 命令行模式：在普通模式下，输入 : 即可转到该模式

光标

上移：k

下移：j

左移：h 或 BS

右移：l 或 <SPA>

移动到行首：0

移动到行尾：$

向上翻页：CTRL-B，注意和 emacs 冲突

向下翻页：CTRL-F，注意和 emacs 冲突

关于光标的上、下、左、右、行首、行尾的移动，还是去使用 emacs 的快捷键，其他的可以参考 vim

右移 1 个单词：w；向右移动 2 个单词：2w；向右移动 5 个单词：5w

左移 1 个单词：b；向左移动 2 个单词：2b；

因为 mac 里的 option 不太好改键，导致很多关于 option(alt) 的快捷键只有特殊字符

向右移动到第 2 个单词的末尾：2e；向右移动到第 5 个单词的末尾：5e

移动到文件第一行：gg

移动到文件最后一行：G

跳转到第 520 行：520 G，一般配合 CTRL-G 文件定位信息

移动到与左（右）括号匹配的右（左）括号：%

进入与退出

无改动退出：<ESC> 后 :q! <Enter>

保存退出：<ESC> 后 :wq <Enter>

文本编辑

删除一个字符：x

删除一个单词：dw，从光标到下一个单词起始处，不包括第一个字符；de，包括单词后的空格都会删掉

删除两个单词：d2w

删除光标后：d$，一行中光标之后的内容被删掉，包括光标处的字符

删除一整行：dd，dd O 常放在一起操作

删除两行：2dd

添加：i，在光标之前插入文本

添加：a，在光标之后插入文本

添加：A，在光标所在行之后添加文本

添加：o，在光标所在行的下一行插入

添加：O，在光标所在行的上一行插入

撤销：u，撤销最后一次命令（向后回溯）

行恢复：U，将整行恢复到原始状态

撤销以前的撤销命令：CTRL-R（向前穿越）

复制：y，yw 复制一个单词，配合 v 进入视图模式配合光标移动实现文本复制

粘贴：p，如果删除的是一行的一部分，则将删除内容置入光标之后；如果删除的是一整行，则将最后一次删除的内容置入光标的下一行

替换一个字符：r，按下 r 后输入字符，即可替换光标处的字符

替换多个字符：R，进入替换模式，可以理解为连续替换

替换单词部分内容：ce，此时删除了单词的光标及其之后的部分，此时也进入到了插入模式

替换整个单词：cw，光标所在的整个单词被删掉

替换句子：c$，删掉一行的光标及其之后的部分

替换一行中的所有指定文本：:s/old/new/g，将光标所在行中所有的 old 替换为 new

替换指定两行中间的所有指定文本：:#,#s/old/new/g，比如，替换 10 行到 15 行及其中间的所有行中的 hello 为 bye，就可以写为 :10,15s/hello/bye/g

替换整个文件中的指定文本：:%s/old/new/g

找到整个文件中的指定文本，并询问是否修改：:%s/old/new/gc

文件

查看光标所在位置及文件状态信息：CTRL-G

文件保存：:w FILENAME，将当前 vim 中正在编辑的文件保存到名为 FILENAME 的文件中

查找

查找文本：/，按下 / 时进入到命令行模式，直接输入要查找的内容即可，从上往下找

​ 向下查找：n

​ 向上查找：N

逆向查找文本：?，同理，但是从下往上找，所以 n 和 N 也反过来了

​ 向下查找：N

​ 向上查找：n

外部命令

临时执行 vim 之外的命令：:!，比如，我想临时查看当前目录的文件都有什么，可以写为 :!ls

帮助

忘了快捷键怎么办：:help w、:help insert-index

自动补全：<TAB> 或 CTRL-D
